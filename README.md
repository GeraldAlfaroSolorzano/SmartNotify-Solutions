# SmartNotify Solutions/Lab Tipos de Comunicacion

Aplicacion web con distintos tipos de comunicacion y para reportes por medio de correos interactivos.

## Tecnologias

- Node.js
- Express
- React
- Vite
- Bootstrap
- MySQL
- Docker
- Nodemailer
- Socket.IO

## Tipos de comunicacion

- HTTP tradicional
- Fetch
- Polling
- Long polling
- Server-Sent Events
- WebSockets
